#  HR Analytics Dashboard – Excel Project

##  Project Overview

This project presents an interactive **HR Analytics Dashboard** built using Microsoft Excel to analyze workforce data and generate actionable business insights.

The project follows a complete data analysis workflow:
Raw Data → Data Cleaning → Data Modeling → Dashboard Development → Business Insights

---

##  Business Problem

HR departments often struggle to:

- Identify reasons behind employee attrition  
- Monitor department-wise workforce distribution  
- Analyze salary and experience patterns  
- Evaluate recruitment source effectiveness  

This dashboard helps HR teams track key metrics and make data-driven decisions.

---

##  Project Files

- `HR_rawdataset.xlsx` – Original raw dataset  
- `HR_cleandataset.xlsx` – Cleaned and processed dataset  
- `HR_Dashboard.xlsx` – Final interactive dashboard  
- `dashboard.png` – Dashboard preview image  

---

##  Project Workflow

1. Data Collection (Raw HR Dataset)
2. Data Cleaning (Removed duplicates, handled missing values)
3. Data Transformation & Structuring
4. KPI Identification
5. Pivot Table & Chart Creation
6. Interactive Dashboard Development using Slicers

---

##  Key Performance Indicators (KPIs)

- **Total Employees:** 311  
- **Current Employees:** 207  
- **Attrition Rate:** 33%  
- **Average Salary:** 69,021  
- **Average Age:** 46  
- **Average Experience:** 12 Years  

---

##  Dashboard Features

- Department-wise Employee Distribution  
- Gender & Marital Status Analysis  
- Department-wise Salary Trends  
- Absenteeism by Department  
- Recruitment Source Analysis  
- Attrition by Department & Manager  
- Employee Satisfaction Score  
- Interactive Filtering using Slicers  

---

##  Key Insights

- Production department shows higher attrition levels.
- LinkedIn is a major recruitment source.
- Certain managers have comparatively higher team attrition.
- IT/IS department reflects higher absenteeism.
- Marital status impacts employee retention trends.
